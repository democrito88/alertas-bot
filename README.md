Projeto para emitir alertas automáticos via telegram

### Uso
Instale as bibliotecas e dependências:
```composer install```

Configure o Token do robô no .env

Execute a migração:
```composer migrate```

...O seed:
```composer seed```

...e envie o relatório do GLPI
```composer enviar```
